# Phone Lookup Web App

A standalone web application for searching phone number details with history and statistics.

## Features
- **Phone Lookup**: Retrieve country and carrier details for any phone number.
- **Search History**: View a log of recent searches.
- **System Analytics**: Track total searches and unique numbers.
- **API Access**: Simple JSON API for programmatic lookups.

## Deployment
This app is ready for deployment on Render, Railway, or any Python-compatible host.
- **Start Command**: `python app.py`
- **Port**: 3000 (for AI Studio) or 8080 (default)
